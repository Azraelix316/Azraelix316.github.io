Installing the Tools

Tags: #frc #coding #to-code-a-robot
Date: 2025-06-05

WPIlib

The most important tool to download is WPIlib's IDE (Integrated Development Environment) so you can actually write code on a robot.
WPIlib download
After downloading the ISO file, right click it to mount the disk image. Then, run the WPIlib Installer. You may need to run as administrator or unblock it via "Properties" if Windows Defender "protects" your computer. Then, follow the instructions in the installer, but make sure to install VS code along with the rest of the tools. The installer will extract and help download the rest of the files.
After following these steps, you should be able to find and run WPIlib VS Code 2025, which is the IDE you'll be using. While you can use other IDEs like Neovim, it's highly recommended to use the bundled VS code for ease of use and to avoid issues later on.

PathPlanner

PathPlanner is a tool used to draw and plan out autonomous paths and implement them in code easily, it's vital for anything related to autonomous - you can download it off the Microsoft Store here.

Phoenix Tuner X

Phoenix Tuner X is a tuning software to help tune and debug motors. You can download this from the Microsoft Store. Latest firmware for specific motors can be found on their GitHub page.

REVlib Hardware Manager

The REVlib Hardware Manager is used to debug and flash NEO Brushless motors - you can download it here.

Git

Follow this tutorial or just click the Git icon in the sidebar and download from there.

GitHub

You should make an account on GitHub as well.
