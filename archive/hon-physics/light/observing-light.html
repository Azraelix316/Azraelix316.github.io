Observing Light

#physics #light

Color

  • Three ways to observe light
    • Absorption
      • Light is absorbed into object object is NOT that color
    • Reflection
      • Light reflects off object object is that color
    • Transmission
      • Light allowed to go through object object is that color
  • Color cones in eyes allow us to detect light
    • Long, medium, and short wavelengths
      • Long (red) cone is sensitive to blue as well which lets us see purple
    • Colorblindness
      • Color receptors dysfunctional
        • Monochromacy
          • No color cones (1 in 30,000 people)
        • Anomalous Trichromacy
          • One color cone messed up
            • 6.3% of men
              • Why jard is colorblind
            • 0.37% of women

Sight

  • Eye has two lenses
    • Cornea - converging lens
      • Convex
    • Lens - also converging lens!
      • Adjusts the image distance
      • Shortsighted is when this is too elongated, creating a poorly focused image
      • Image forms behind retina
  • Farsightedness
    • Converging lens to form the image closer
  • Nearsighted
    • Diverging lens to form the image further away
  • Myopia
  • Presbyopia
  • Astigmatism
    • Different parts of lens/cornea forms different images
