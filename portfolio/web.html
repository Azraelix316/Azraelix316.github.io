A short foray into web design, I went through a lot of iterations and really developed my artistic design skills and my coding skills. I also utilized my past p5.js projects to provide some basic animations and backgrounds, to make the website more visually diverse. The things I'm most proud of on this website are the UX for the menu, which I happened to somehow create, and also the implementation of p5.js as well - now, I can integrate p5.js into actual web applications and this really brought me into JS for it's true purpose of web development.
