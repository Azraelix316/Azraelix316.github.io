Math Modeling

Math Modeling

For math modeling, I did a lot of work with C++, especially low level C++, to optimize algorithms. I created a library, DSCILIB, currently still a WIP but with many helpful helper functions in order to help make coding more efficient - I learned some more advanced mathematics in linear algebra such as PCA and also transition matrices specifically for this library, which I used to help accelerate my workflow during the HiMCM 2025 competition. It's still a work in progress, but I used many of the matrix based helper functions and QoL, and hopefully it will develop to become a core library. The actual usage of this library was for the MCM/HiMCM math modeling competitions, where much of the work was creating simulations to accumulate data that could be used.
