    • Concave is caved in
      • One ray parallel to the principal axis which will reflect through the focus
      • One ray through the object and principal focus which will reflect and create a ray parallel to the principal axis
        • In the case of the object being closer than the focus, draw from the focus to the object
      • 3 main cases
        • Object is behind focus
        • Object is between focus and center of curvature
        • Object is in front of focus
    • Convex is facing outwards (i.e not caved in)
      • One ray parallel to the principal axis that reflects off and seems to intersect the focus
