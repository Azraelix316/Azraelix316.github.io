Polaroid Filters

#physics #light

  • Limit light to only one plane (Polarization)
  • Invented by Edwin hall
    • Broke into a physics lab after dropping out of Harvard
    • Created using polymer acting like walls that prevent movement
    • See figure
  • Extremely useful for a variety of things
    • Can fix glare
      • after reflection, glare is typically horizontal
      • So polaroid filter can filter all of the light and prevent it
    • LCD screen
      • Electrically controlled polaroid filters (Liquid Diode Crystals) that polarizes and can block or unblock specific pixels
      • Via double polaroid filters, you can block ALL the light if you filter horizontal and vertical
        • Light can only pass through one filter but not through both
    • Privacy screen
      • 45 degree polaroid filter that splits light into components to get past the filters
        • 3 layers of filters, middle one is 45 degrees that reorients the light
        • Allows some light through, blocks all when the middle polaroid filter is removed
        • The light ends up being half as intense normally
  • 3D glasses
    • One vertical filter one horizontal filter
      • Forces eyes to create the 3D image from two separate images
    • Used to be color filters
